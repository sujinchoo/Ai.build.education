# AI Service Development Curriculum PRD

본 문서는 AI 서비스 개발 교육 커리큘럼의 설계 문서입니다.

------------------------------------------------------------------------

# Product Overview

이 프로젝트는 초급자가 AI 서비스를 직접 개발할 수 있도록 돕는 교육
플랫폼 콘텐츠입니다.

강의는 슬라이드 기반 웹페이지 형식으로 제공되며\
GitHub 정적 페이지 형태로 배포됩니다.

------------------------------------------------------------------------

# Problem

많은 사람들이 AI 서비스 개발에 관심이 있지만 다음과 같은 문제를
겪습니다.

-   프로그래밍을 어디서부터 배워야 할지 모름
-   AI 기술을 실제 서비스로 연결하는 방법을 모름
-   아이디어는 있지만 구현 방법을 모름

------------------------------------------------------------------------

# Solution

본 커리큘럼은 다음 흐름으로 문제를 해결합니다.

아이디어\
↓\
프로그래밍 기초\
↓\
웹 구조 이해\
↓\
AI 기술 이해\
↓\
AI 서비스 구현\
↓\
배포 및 운영
