# AI Builder Curriculum

문과생을 위한 AI 제작자 양성 커리큘럼입니다.

## 🎯 교육 목표

-   AI의 구조를 이해한다.
-   머신러닝과 딥러닝을 구현한다.
-   AI 서비스를 설계한다.
-   GitHub Static Page로 배포한다.

------------------------------------------------------------------------

## 🧭 3단계 설계

### 1단계: 이해 (Understand)

AI의 철학, 머신러닝의 구조, 딥러닝의 원리를 학습합니다.

### 2단계: 구현 (Build)

Python 기반 머신러닝 및 딥러닝 실습을 진행합니다.

### 3단계: 배포 (Deploy)

GitHub에 코드를 업로드하고 Static Page로 공개합니다.

------------------------------------------------------------------------

## 📦 모듈 구성

-   Module 1: AI 사고의 기초
-   Module 2: 머신러닝 구조 이해
-   Module 3: 딥러닝과 LLM
-   Module 4: 서비스 설계
-   Module 5: 배포 실전

------------------------------------------------------------------------

## 🚀 사용 방법

1.  이 저장소를 Fork 합니다.
2.  INDEX.html을 GitHub Pages로 배포합니다.
3.  각 모듈을 확장하여 서비스 프로젝트를 완성합니다.

------------------------------------------------------------------------
