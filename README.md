# AI Service Development Curriculum

AI 서비스 개발을 목표로 하는 **12주 교육 커리큘럼 프로젝트**입니다.  
초급자도 **아이디어 → 코드 → 서비스 배포**까지 경험할 수 있도록 설계된 실습 중심 학습 자료입니다.

이 저장소는 강의 슬라이드형 웹페이지와 교육 콘텐츠를 포함하며  
GitHub Pages 또는 정적 사이트로 배포하여 학습 자료로 활용할 수 있습니다.

---

# 프로젝트 목표

이 프로젝트의 목표는 다음과 같습니다.

- 컴퓨터의 기본 원리와 역사 이해
- Python 프로그래밍 기초 습득
- 웹 개발 구조 이해 (HTML / CSS / 서버 구조)
- AI 및 LLM 기반 서비스 구조 이해
- GitHub 기반 협업 및 버전 관리 경험
- AI 서비스를 실제 웹앱 형태로 배포

최종적으로 학습자는 **AI 기능이 포함된 웹 서비스 MVP를 직접 구현**하게 됩니다.

---

# 학습 결과 (Learning Outcome)

이 커리큘럼을 완료하면 다음을 수행할 수 있습니다.

- 컴퓨터가 데이터를 처리하는 기본 구조 설명
- Python 문법을 이용해 간단한 프로그램 작성
- 웹페이지 구조 이해 및 간단한 UI 구현
- AI API를 활용한 서비스 기능 구현
- GitHub을 활용한 프로젝트 관리
- 클라우드 플랫폼을 이용한 서비스 배포

---

# 커리큘럼 구조

전체 과정은 4단계로 구성됩니다.

## 1단계 — 개발 기초

- 컴퓨터 원리 및 프로그래밍 기초
- Python 문법
- HTML / CSS
- 서버 구조 이해

## 2단계 — AI / 머신러닝 이해

- 머신러닝 개념
- 신경망 기본 구조
- 딥러닝 모델 구조 이해

## 3단계 — LLM 기반 서비스

- Transformer 구조
- LLM 동작 방식
- API 기반 AI 서비스

## 4단계 — 서비스 구현

- GitHub 기반 프로젝트 관리
- AI 서비스 MVP 개발
- 클라우드 배포
- 서비스 운영
