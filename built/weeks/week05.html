Week 05 — Neural Network 기초

주제

퍼셉트론과 다층 신경망 구조, 활성화 함수, 학습 과정을 이해한다.

비주얼 콘셉트

텍스트 흐름

입력층 → 은닉층(활성화 함수) → 출력층 → 손실 계산 → 역전파로 가중치 업데이트

그림

flowchart LR I[입력층] --> H[은닉층 ReLU/Sigmoid] H --> O[출력층] O --> L[손실 계산] L --> B[역전파] B --> W[가중치 업데이트] W --> H

학습 목표

  • 퍼셉트론 구조 이해
  • ReLU/Sigmoid/Softmax 역할 구분
  • 오차역전파 개념 이해

실습 미션

간단한 MLPClassifier를 이용해 이진 분류 실습 후 정확도 확인.
