Week 09 — LLM 기초: GPT, BERT, 생성형 AI의 동작 원리

학습 목표

  • LLM의 사전학습(Pretraining)과 파인튜닝(Fine-tuning)을 구분한다.
  • GPT와 BERT의 목적 차이를 설명한다.
  • 토큰화, 확률적 다음 토큰 예측 원리를 이해한다.

1. LLM이란?

대규모 텍스트 데이터와 Transformer로 학습한 언어 모델.

2. 학습 단계

  1. 사전학습: 방대한 텍스트로 일반 언어 패턴 학습
  2. 지시튜닝/파인튜닝: 특정 목적에 맞게 조정
  3. 정렬(Alignment): 사람 의도에 맞는 응답 강화(RLHF 등)

3. GPT vs BERT

모델방향성주 용도
GPTDecoder 기반(자기회귀)텍스트 생성, 챗봇
BERTEncoder 기반(양방향)이해/분류/검색

4. 생성 원리

  • 입력을 토큰으로 분해
  • 각 단계에서 다음 토큰 확률 계산
  • 샘플링(temperature, top-k/top-p)으로 출력 다양성 제어

5. 한계와 주의점

  • 환각(Hallucination)
  • 최신성 부족(학습 시점 제한)
  • 편향/보안/저작권 이슈

실습 미션

  1. temperature 변화에 따른 답변 스타일 비교.
  2. 동일 질문에서 GPT/BERT 계열 역할 분리 설계.
  3. 프롬프트 개선 전/후 품질 평가표 작성.

정리

LLM은 확률적 언어 생성기이며, 좋은 활용에는 프롬프트와 검증 절차가 필수다.
