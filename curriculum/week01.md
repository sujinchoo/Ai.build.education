# Week 01 — 컴퓨터 원리와 Python 입문

## 주제
컴퓨터의 기본 동작 원리와 컴퓨팅 역사, 그리고 Python 실행 기초를 연결해 이해한다.

---

## 학습 목표
- 컴퓨터의 4대 기능(입력/처리/저장/출력)을 설명할 수 있다.
- 튜링 머신과 폰 노이만 구조가 현대 컴퓨터에 준 영향을 설명할 수 있다.
- Python 실행 구조, 변수, 기본 자료형을 사용해 간단한 코드를 작성할 수 있다.

---

## 비주얼 콘셉트
### 텍스트 흐름
입력 장치 → CPU 처리 → 메모리 저장 → 출력 장치

튜링 머신 개념 → 폰 노이만 구조 → 현대 범용 컴퓨팅

### 그림
```mermaid
flowchart LR

    C --> M[저장: 메모리/스토리지]
    M --> O[출력: 모니터/스피커]
```

```mermaid
flowchart LR
    T[튜링 머신 1936] --> V[폰 노이만 구조 1945]

```

---

## 학습내용
- 컴퓨터는 **입력-처리-저장-출력**의 순환 구조로 동작한다.
- 현대 소프트웨어는 폰 노이만 구조(메모리에 코드와 데이터를 함께 저장) 위에서 실행된다.
- Python은 인터프리터 언어로, 작성한 코드를 위에서 아래로 순차 실행한다.

```python
user_name = "Mina"
score = 95
print(user_name, score)
print(type(user_name), type(score))
```

- 위 코드는 문자열(`str`)과 정수(`int`) 자료형을 출력하고 타입을 확인한다.
- 최신 개발 환경에서는 로컬 IDE + 클라우드 실행 환경(예: Codespaces, Colab)을 함께 사용한다.
