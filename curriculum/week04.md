# Week 04 — 머신러닝 기초: 개념, 원리, 대표 알고리즘

## 학습 목표
- 머신러닝의 정의와 학습 패러다임을 구분한다.
- 지도/비지도/강화학습의 차이를 설명한다.
- 과적합, 일반화, 평가지표의 의미를 이해한다.

---

## 1. 머신러닝이란?
명시적 규칙 대신 데이터에서 패턴을 학습해 예측/분류를 수행하는 방법.

## 2. 학습 유형
| 유형 | 목표 | 대표 알고리즘 |
|---|---|---|
| 지도학습 | 정답 라벨 예측 | 선형회귀, 로지스틱회귀, SVM, 트리 |
| 비지도학습 | 구조/군집 발견 | K-means, PCA |
| 강화학습 | 보상 최대화 정책 학습 | Q-learning, Policy Gradient |

## 3. 핵심 원리
- 손실 함수(Loss): 오차를 수치화
- 최적화(Optimization): 손실 최소화
- 일반화(Generalization): 새로운 데이터에서도 성능 유지

## 4. 평가 지표
- 회귀: MAE, MSE, RMSE
- 분류: Accuracy, Precision, Recall, F1, ROC-AUC

## 5. 과적합/과소적합
- 과적합: 훈련 데이터에만 과도 적응
- 해결: 규제(L1/L2), 데이터 증강, 교차검증, 단순한 모델

## 실습 미션
1. 학습 곡선으로 과적합 여부 판단.
2. 동일 데이터에 2개 알고리즘 적용 후 성능 비교.
3. confusion matrix 해석 문서 작성.

## 정리
머신러닝은 "데이터 + 수학적 최적화"의 조합이며, 평가와 일반화가 핵심이다.
